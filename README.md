# Projet de Recherche Opérationnelle 2026

Ce projet présente la résolution de trois problèmes d'optimisation en Recherche Opérationnelle, dont deux problèmes réels liés à la Mauritanie et un problème classique mondialement connu.

## Sujets choisis

1. Transport de poissons entre Nouadhibou et Nouakchott
2. Routage des camions d'eau dans les quartiers périphériques
3. Problème du voyageur de commerce, TSP

## Objectif du projet

L'objectif est de modéliser, résoudre et interpréter des problèmes d'optimisation réels en utilisant des outils de Recherche Opérationnelle.

Chaque sujet contient :

- Une description du problème
- Une formulation mathématique
- Un exercice lié à la méthode du simplexe
- Un code Python de résolution
- Une interprétation des résultats

## Outils utilisés

- Python
- Pyomo
- OR-Tools
- GLPK
- Overleaf / LaTeX
- GitHub
- PowerPoint
